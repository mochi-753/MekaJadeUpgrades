### Description

This is an **unofficial 1.20.1 port** of the **MekaJadeUpgrades** mod.
This addon bridges Mekanism and Jade, adding extra tooltips to the Jade HUD that display the installed upgrades inside
Mekanism machines.

### Disclaimer

**This is an UNOFFICIAL port.** Please **DO NOT** report any bugs, issues, or crashes related to this 1.20.1 version to
the original author.

### Features

* Displays installed Mekanism upgrades directly in the Jade tooltip.
* Quickly check machine status without opening the GUI.
